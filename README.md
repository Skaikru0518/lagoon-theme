# Lagoon

Aesthetic teal-to-lavender VS Code theme with two variants:

- **Lagoon Dark** — deep teal-black background, mint-teal keywords, cyan functions, lavender methods, teal-green strings/numbers, italic comments.
- **Lagoon Light** — airy teal background, saturated tokens in the same hue family.

## Activate

`Ctrl+K Ctrl+T` → pick **Lagoon Dark** or **Lagoon Light**.

## Palette

| Token    | Dark      | Light     |
|----------|-----------|-----------|
| Keyword  | `#4DBDA8` | `#0E8C7A` |
| Function | `#5BA8E0` | `#2476B8` |
| Method   | `#A99BE6` | `#7C5CD4` |
| String   | `#67C7A3` | `#1E9E78` |
| Comment  | `#56676B` | `#8AA0A0` |
